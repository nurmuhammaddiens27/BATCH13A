Menghitung Luas dan Keliling Lingkaran

Algoritma

Luas

  1. Masukan nilai variabel jari-jari
  2. Menghitung luas lingkaran
  3. Rumus : 3.14 x jari-jari x jari-jari
  4. Masukan nilai jari-jari
  5. Menampilkan hasil dari perhitungan luas lingkaran

Keliling

  1. Masukan nilai variabel jari-jari
  2. Menghitung keliling lingkaran
  3. Rumus : 2 x 3.14 x jari-jari
  4. Masukan nilai jari-jari
  5. Menampilkan hasil dari perhitungan keliling lingkaran

Pseudocode

luas

  2. Menghitung luas lingkaran
  3. Rumus : 2 x 3.14 x jari-jari
